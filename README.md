# File Management GUI

A lightweight desktop utility for common file-organization tasks, built with **PyQt6**. It provides a simple, guided interface for copying files, applying consistent naming conventions, and sorting items into type-based folders.

## Key Features
- **Copy files** to a chosen destination.
- **Rename files** with custom prefixes for consistent naming.
- **Sort by file type** into automatically created subfolders.

## Getting Started
### Prerequisites
- **Python 3.6+**
- **PyQt6**

### Run the App
```bash
python -m finalproject.main
```

## Typical Workflow
1. Launch the application.
2. Choose the action you want to perform (copy, rename, or sort).
3. Follow the prompts in the GUI to select files/folders and complete the task.

## Project Structure
- `finalproject/` — application source code.
- `finalproject.main` — entry point for the GUI.

## Notes
This project is intended as a simple, approachable utility for everyday file management tasks.
